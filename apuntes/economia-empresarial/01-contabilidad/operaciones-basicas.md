# Operaciones básicas

Una empresa básicamente **compra** bienes, **paga** por lo que compra, **vende** y **cobra**
por lo que vende. Acá se registran algunos movimientos típicos en el Libro Diario aplicando
la [[partida-doble]].

> Convención de notación: `(A+)` = aumento de Activo, `(P+)` = aumento de Pasivo,
> `(PN+)` = aumento de Patrimonio Neto, `(A−)` = disminución de Activo, etc.
> En el libro diario, **la cuenta que va al principio es la que recibe el flujo de cash**;
> las cuentas que empiezan con "a" están acreditadas (van al Haber).

## Asientos típicos

| Cuenta | Debe | Haber |
|--------|-----:|------:|
| Caja y Bancos (A+) | 5000 | |
| &nbsp;&nbsp;a Capital Social (PN+) | | 5000 |
| Bienes de Cambio (A+) | 100 | |
| &nbsp;&nbsp;a Deudas Corrientes (P+) | | 100 |
| Bienes de Uso (A+) | 1000 | |
| &nbsp;&nbsp;a Caja (A−) | | 1000 |
| Deudas Bancarias de Corto Plazo (P−) | 500 | |
| &nbsp;&nbsp;a Caja y Bancos (A−) | | 500 |

### Explicación de cada operación

1. **Aporte de capital de los socios.** Entra cash → Caja (Activo) aumenta → se **debita**.
   Capital Social (PN) aumenta → se **acredita** (Pasivo y PN funcionan al revés que Activo).
2. **Compra de bienes de cambio a crédito.** Bienes de Cambio (Activo) aumenta → se debita.
   Deudas Corrientes con proveedores (Pasivo) aumenta → se acredita.
3. **Compra de una máquina (bien de uso) con cash.** Bienes de Uso (Activo) aumenta → se
   debita. Caja (Activo) disminuye → se acredita. (Es un movimiento dentro del Activo: no
   toca el cuadro de resultados — ver [[cuadro-de-resultados]].)
4. **Pago de amortización de capital de una deuda bancaria.** Deudas Bancarias (Pasivo)
   disminuye → se debita. Caja (Activo) disminuye → se acredita.
