# Punto de equilibrio

Volumen de producción/venta en el que los **ingresos totales igualan a los costos totales**
(fijos + variables): el beneficio es cero, la empresa no gana ni pierde. Por encima de ese
punto hay beneficios; por debajo, pérdidas.

Sirve para cuantificar el volumen mínimo de unidades que hay que producir y vender para
empezar a tener ganancias.

## Fórmula

Bajo el supuesto de **precio de venta constante** (independiente de la cantidad):

```
(1) Ingreso total:  IT = p · q
(2) Costo total:    CT = cv · q + CF
```

donde `p` = precio unitario, `cv` = costo variable unitario, `CF` = costos fijos.
Se espera `cv < p` para que la empresa pueda funcionar.

Igualando IT = CT y despejando q:

```
p · q = cv · q + CF

q* = CF / (p − cv)
```

## Gráfico

```
$ │              IT
  │            /
  │          / ___ CT
  │        /  /
  │      / */          * = punto de equilibrio (q*)
  │    / /
  │  /─/──────────── CF
  │ //
  └──────────────── q
      q*
   ←perdida→│←─ beneficio ─→
```

A la izquierda de `q*` la empresa pierde dinero; a la derecha, gana.

## Interpretación

Cuanto **más bajo** sea el punto de equilibrio, menos probable es perder dinero ante
fluctuaciones del mercado. Con precio constante, si se reduce el `q*` (bajando CF o cv) se
obtiene más utilidad para todos los volúmenes por encima del nuevo punto de equilibrio.
